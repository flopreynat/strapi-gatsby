# Strapi and Gatsby

Simply having fun with strapi and Gatsby

a few useful commandlines and urls:

## Strapi
- cd strapi/my-project
- yarn develop
- http://localhost:1337/
- http://localhost:1337/admin/
- http://localhost:1337/restaurants

## Gatsby
- cd gatsby/website
- gatsby develop
